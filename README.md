# 👕 AI Virtual Try-On System

## 📌 Project Overview
This project is an AI-based Virtual Try-On System that allows users to try clothes virtually using images or live camera. It uses computer vision techniques to overlay garments on the human body.

---

## 🚀 Features
- Upload image or use live camera
- Detect human body using pose estimation
- Overlay clothes on user body
- Real-time preview (basic)
- Privacy-focused (no data storage)

---

## ⚙️ Tech Stack
### Frontend:
- Next.js
- Tailwind CSS

### Backend:
- FastAPI
- OpenCV
- MediaPipe

---

## 🧠 Methodology
1. Capture image/video
2. Detect body keypoints using MediaPipe
3. Process image using OpenCV
4. Overlay clothing image on detected body
5. Display result on frontend

---

## 🎯 Objectives
- Provide realistic virtual try-on
- Improve online shopping experience
- Maintain user privacy
